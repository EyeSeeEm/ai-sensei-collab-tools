ü§ñ OpenClaw on VPS

Self-hosted bot on a dedicated server. Accessible via Telegram, Discord, WhatsApp, Slack, and Email (via Gmail MCP). Can have its own email address (e.g., sensei@aisensei.com).

Shared context - everyone talks to the same bot with the same knowledge base. Full customization: system prompts, MCP tools, web search, file generation.

Downside: Requires initial setup (2-4 hours) and occasional maintenance. Community support only.

Best for
Team collaboration where everyone needs access via their preferred chat app. Marketing content, strategy discussions, shared AI assistant with persistent knowledge.

üí¨ Claude for Slack

Official Anthropic integration. @mention in channels. Auto-routes coding tasks to Claude Code web sessions with GitHub PR creation.

Has web search and MCP tools (if configured on claude.ai). Searches your Slack workspace for context.

Downside: No persistent memory for non-code work - each thread starts fresh. Per-user pricing. Requires Slack workspace.

Best for
Teams already on Slack who need code collaboration with GitHub PRs. Good for dev work, less ideal for marketing/content.

Best for
Individual coding and power-user work. Max's personal development. Not for team collaboration.
